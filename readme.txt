eBooks library plugin permits to add books from the backend and helps users/visitors to search for books in the library. The reader can search books by author, publisher, price, and ratings. They can also search for their favorite books by their name.

== Description ==

eBooks library is the best Books Showcase Plugin which will build a beautiful mobile-friendly book store or a books library.
Create your online ebook store where you can display your books via shortcode and let the visitors search for their favorite book and enjoy reading.

To display book search filter, use the shortcode [mdlibsfilter] 

FEATURES OF eBooks Library Plugin:
* Add unlimited books to the library.
* Search book by its name.
* Filter the books by title, author, publisher, price, and ratings.
* Display search filter anywhere on page or post via shortcode

== Installation ==

Installation Steps

1. Upload the folder to `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Goto the `eBooks` admin Menu and add books.

== Frequently Asked Questions ==

= How to display book search filter? =

Copy and paste the shortcode <code>[mdlibsfilter]</code> in your post/pages.

= How to add books in library =

After activating the plugin, you will see “eBooks” in WordPress Admin Dashboard.

Go to “Add New Book” and add your Books as many as you want.
